# Raw Data

The raw datasets used in this project are stored externally on Hugging Face:

**ringoch/nyc-commercial-data**

They are not kept directly in this GitHub repository in order to reduce repository size and keep version control clean.

This GitHub repo focuses on:
- processed datasets in `data/processed/`
- analysis, modeling, and application code

If reprocessing is needed, please retrieve the original source files from the Hugging Face raw data repository above.
